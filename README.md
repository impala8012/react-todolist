# React Todo-list

### [連結](https://impala8012.github.io/react-todolist/)
![](https://i.imgur.com/6JExQ6e.png)

## 功能介紹

- 新增 todo
- 刪除 todo
- 編輯 todo
- 標記已完成 / 未完成，其中已完成的 todo 會畫線
- 清空全部 todo
- 篩選 todo，分別顯示全部、未完成、已完成的 todo

![](https://i.imgur.com/ZDIsZnT.gif)

## 使用技術與工具
前端套件
- React Hooks 
- styled-component
